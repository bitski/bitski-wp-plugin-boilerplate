## 📋 Requirements

### Server

- PHP 8.1+ (tested up to PHP 8.4)
- WordPress 6.5+ (Full Site Editing ready)
- MySQL 5.7+ / MariaDB 10.4+

### Development

- Composer 2.7+
- Node.js 18+ / 20 LTS
- npm 9+ / yarn 1.22+
- sass, esbuild
- PHPCS 3.8+ (phpcs.xml inklusive, PSR-12)
